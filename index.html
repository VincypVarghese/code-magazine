The basic languages of the Web:HTML

posted by laura jones on monday,june 21st 2007

All modern websites and webapplications are built using three fundamental technologies:HTML,CSS and Javascript.These are the languages of the website

In this post,let's focus on Html.we will learn what html is all about,and why you too should learn about it

What is html

Html stands for Hyper Text Markup Language.It's markup language that web developers use to structure and describe the content of a webpage(not a programming language).

Html consists of elements that describe the different type of content.Paragraph,links,images,heading,video,etc.Web browsers understand HTML and render HTML code as websites.

In HTML,each element is madeup of 3 parts:

  1. opening tag
  2. closing tag
  3. actual element

You can learn more at MDN Web Docs
